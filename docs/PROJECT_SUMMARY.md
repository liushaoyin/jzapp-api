### 技术栈
- **前端框架**: uni-app + Vue.js
- **UI 组件**: uni-ui
- **图表库**: uCharts
- **后端服务**: uniCloud
- **数据库**: MongoDB Atlas
- **部署平台**: Vercel + GitHub Pages

### 功能特性
- 💰 账单管理（添加、编辑、删除）
- 👥 成员管理（多成员记账）
- 📊 数据统计（可视化图表）
- 🔄 数据同步（云端备份）
- 📱 跨平台（H5、小程序、App）
- 🎨 现代化UI设计
