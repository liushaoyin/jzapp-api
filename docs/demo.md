## 🛠️ 技术栈

- **前端框架**: uni-app + Vue.js
- **UI 组件**: uni-ui
- **图表库**: uCharts
- **后端服务**: uniCloud
- **数据库**: MongoDB Atlas
- **部署平台**: Vercel

## 📊 性能指标

- 首屏加载时间: < 2s

- 数据同步延迟: < 1s
